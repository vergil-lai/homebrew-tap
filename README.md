# Vergil-lai Tap

## How do I install these formulae?

`brew install vergil-lai/tap/<formula>`

Or `brew tap vergil-lai/tap` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "vergil-lai/tap"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
